# GGUF Model Files

Place `.gguf` model files in this directory for use by the LLM narrator.
GGUF files are not included in the repository due to their large size.

## Recommended Models

A `Q4_K_M` quantization is a good balance between quality and VRAM usage (~6 GB at 8B).

- **Llama 3.1 8B Instruct** (quality) — https://huggingface.co/bartowski/Meta-Llama-3.1-8B-Instruct-GGUF
- **Qwen2.5 7B Instruct** (strong JSON/structured output) — https://huggingface.co/bartowski/Qwen2.5-7B-Instruct-GGUF
- **Llama 3.2 3B Instruct** (latency-first, ~3 GB VRAM) — https://huggingface.co/bartowski/Llama-3.2-3B-Instruct-GGUF

## Placement

Download the desired `.gguf` file and place it directly in this `models/` directory.
The narrator will auto-discover all `*.gguf` files present here at startup.
